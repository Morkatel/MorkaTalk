# MorkaTalk

# Project Overview
MorkaTalk is a World of Warcraft addon designed to enhance accessibility by providing text-to-speech (TTS) functionality for in-game UI elements containing text. It reads out quests, NPC dialogues, auction house items, merchant prices, and error messages, making the game more accessible for visually impaired players.
